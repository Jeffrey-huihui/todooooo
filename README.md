# Todooooo - 待办事项应用

一个简洁美观的桌面待办事项应用，支持窗口置顶功能。

## 功能特点

✅ **添加待办事项** - 快速添加新任务
✅ **标记完成** - 点击复选框标记任务完成状态
✅ **编辑任务** - 点击编辑按钮修改任务内容
✅ **删除任务** - 点击删除按钮移除任务
✅ **任务优先级** - 设置任务优先级（高/中/低），自动排序
✅ **截止日期** - 设置任务截止日期，自动显示到期状态
✅ **到期提醒** - 任务到期前1小时和过期后自动提醒
✅ **二级事项** - 为任务添加子任务，细分任务内容
✅ **窗口置顶** - 将窗口固定在最上层，方便查看
✅ **本地存储** - 数据保存在本地 JSON 文件中
✅ **统计信息** - 显示总任务数、已完成数和待完成数
✅ **美观界面** - 现代化 UI 设计，操作流畅

## 安装运行

1. 安装依赖：
```bash
npm install
```

2. 运行应用：
```bash
npm start
```

3. 开发模式（带开发者工具）：
```bash
npm run dev
```

## 快捷键

- `Ctrl + N` (或 `Cmd + N`) - 聚焦到输入框
- `Enter` - 添加新任务
- `Enter` (编辑模式) - 保存编辑
- `Esc` (编辑模式) - 取消编辑

## 功能说明

### 优先级
- **高优先级** - 红色标记，优先显示
- **中优先级** - 橙色标记，默认优先级
- **低优先级** - 绿色标记，靠后显示

任务列表按优先级自动排序：高 > 中 > 低

### 截止日期与提醒
- 可以设置每个任务的截止日期和时间
- 系统每分钟自动检查到期任务
- 到期前1小时发送提醒通知
- 过期任务会显示过期标记
- 日期显示格式：
  - "今天" - 今天的任务
  - "明天" - 明天的任务
  - "月/日" - 其他日期的任务

### 子任务
- 每个任务可以添加多个子任务
- 子任务可以独立标记完成
- 显示子任务完成进度（如：子任务 2/5）
- 支持编辑和删除子任务

## 数据存储

数据保存在项目目录的 `data/todos.json` 文件中。

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **HTML/CSS/JavaScript** - 纯前端技术
- **Node.js** - 文件系统操作和通知功能
